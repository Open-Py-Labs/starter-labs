# More Learning Resources

Human-curated resources ***\*\*finger guns\*\****


## Python Favorites

- [CodingBat](https://codingbat.com/python) - short, focused coding drills
- [Advent of Code](https://adventofcode.com/events) - annual puzzle series, all years archived
- [Code In Place](https://codeinplace.stanford.edu/) - Stanford's intro to Python with lessons, labs, and a community forum (less boring than other courses in my opinion)
- [Automate the Boring Stuff](https://automatetheboringstuff.com/) - free book, practical real-world projects (though automating boring stuff isn't not boring)

## Python Data, ML and AI Platforms
- [Kaggle](https://www.kaggle.com/) - Datasets, ML competitions and more
- [Zindi](https://zindi.africa/) - African data science platform with competitions, datasets, and more
- [Driven Data](https://www.drivendata.org/) - Health, science and social impact competitions and datasets 

## Cybersecurity Learning Platforms
- [Missionable](https://missionable.github.io) - Cybersecurity portfolio building missions
- [Cylab Cybersecurity Academy](https://cylabacademy.org) - Carnegie Mellon cybersecurity learning with labs and competitions
- [OverTheWire Wargames - Bandit](https://overthewire.org/wargames/bandit/bandit0.html) - a series of progressively harder linux and network security challenges
- [CTFlearn](https://ctflearn.com/) - gamified cybersecurity learning with labs and competitions

## Additional Python learning, but ughhh exhausting
- [Harvard's CS50](https://cs50.harvard.edu/python/) — free intro to Python course from Harvard
- [MIT 6.0001](https://ocw.mit.edu/courses/6-0001-introduction-to-computer-science-and-programming-in-python-fall-2016/) — intro CS using Python, free on MIT OpenCourseWare
