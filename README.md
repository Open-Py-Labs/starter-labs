# 🚀 OpenPyLabs - Starter Labs

[![Use this template](https://img.shields.io/badge/Use%20this%20template-2ea44f?style=for-the-badge&logo=github)](https://github.com/open-py-labs/starter-labs/generate)

Short Python labs for building real-world skills.

| # | Lab | Skills |
|---|-----|--------|
| 01 | [Search Query Builder](./01-search-query-builder/) | strings, f-strings, functions, conditionals |
| 02 | [Duplicate File Finder](./02-duplicate-file-finder/) | file I/O, hashing, dicts |

**How it works:** Open the `starter.py` file in any lab module, build the function, and check your work by running `test.py`.

**Want to run in your browser?** Click "Use this template" above to create your own copy of this repo, then open a *GitHub Codespace* from your repo. Your work saves to your GitHub portfolio. If you're developing locally, just clone your copy and run as normal.

***Want to contribute a tip or a new lab?*** See [CONTRIBUTING.md](./CONTRIBUTING.md).
